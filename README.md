# QA Bootcamp Automation Testing Projects

This repository contains automation testing projects from a Quality Assurance bootcamp. Each session folder (`Sesi9`, `Sesi10`, etc.) represents a different topic in test automation.

## Project Structure

```
e:/bootcamp/
├── Sesi9_AnandaYudhistira/          # Basic Selenium + Mocha
├── Sesi10_AnandaYudhistira/         # Selenium WebDriver tests (Firefox/Chrome)
├── Sesi11_AnandaYudhistira/         # Page Object Model + Visual Testing
├── Sesi13_AnandaYudhistira/         # WebdriverIO + Allure Reports
├── Sesi14_AnandaYudhistira/         # Advanced WebdriverIO + Appium Setup
├── api-test/                        # API Testing (Newman/Postman?)
├── New folder/                      # WebdriverIO demo
├── cypress/                         # Cypress E2E tests
└── Other sessions/files             # API, JMeter, etc.
```
